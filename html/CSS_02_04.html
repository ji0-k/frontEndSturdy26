CSS 선택자는 정말 중요합니다!

다양한 선택자를 배워보겠습니다.

이 부분도 강조되어야 합니다.
  • 전체 선택자
  • 타입 선택자
  • 클래스 선택자
  • ID 선택자


4가지 기본 선택자의 특징

  1. 1. 전체 선택자 (*)/우선순위 가장낮음
    2. - 모든 요소를 선택하는 만능 도구 - 마치 "전교생 집합!"과 같은 개념
  2. 2. 타입 선택자 (태그명)/우선순위 낮음
    3. - 같은 종류의 모든 요소를 선택 - "모든 제목(h1) 집합!" 같은 느낌
  3. 3. 클래스 선택자 (.클래스명)/우선순위 높음
    4. - 같은 클래스를 가진 요소들을 선택 - "축구부 학생들 집합!" 처럼 그룹별 선택
  4. 4. ID 선택자 (#아이디명)/우선순위 가장 높음
    5. - 특정 ID를 가진 하나의 요소만 선택 - "학번 20240001번 학생!" 처럼 개별 지정

선택자들은 구체성 우선순위를가짐
