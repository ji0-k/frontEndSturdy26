오늘의 개발 일기

아침: HTML 공부

오늘은 HTML 태그들을 배웠습니다. 특히 시맨틱 태그의 중요성을 깨달았어요.

hr 태그: 의미 있는 구분선 만들기

hr(Horizontal Rule) 태그는 단순히 선을 그어주는 것이 아니라,
주제의 전환이나 내용의 구분을 의미합니다.
마치 책에서 챕터가 바뀔 때 페이지를 나누는 것처럼, 웹페이지에서도 내용의 흐름이 바뀔 때 사용합니다.

div와 span: 콘텐츠를 담는 그릇

div 태그는 블록 레벨 요소로, 큰 영역을 나누는 컨테이너 역할을 합니다.
마치 방을 칸막이로 나누는 것처럼, 웹페이지의 섹션을 구분할 때 사용해요.

span 태그는 인라인 요소로, 텍스트의 일부분만 감쌀 때 사용합니다.
문장 속에서 특정 단어나 구문을 강조하거나 스타일을 적용할 때 유용해요.

점심: 코딩 실습

배운 내용을 바탕으로 간단한 웹페이지를 만들어봤습니다.

저녁: 복습

하루 종일 배운 내용을 정리하고 내일 계획을 세웠습니다.

HTML 학습 정리

HTML은 웹페이지의 구조를 만드는 언어입니다. 시맨틱 태그를 사용하면 더 의미있는 마크업이 가능해요.

오늘 배운 기술 스택: HTML5, CSS3, JavaScript



구분선과 레이아웃 학습 정리

hr 태그 : 주제전환 이나 내용 구분을 위한 의미론적 구분선
