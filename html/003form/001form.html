- 입력 칸들: 사용자가 정보를 입력하는 공간

< input type="text" name="username">

< /form>
- action: 폼 데이터를 어디로 보낼지 정하는 주소 (목적지)
- method: 데이터를 어떤 방식으로 보낼지 정하는 방법 (GET 또는 POST)

회원가입

- for 속성과 id 속성을 연결하면 라벨을 클릭해도 입력 칸이 선택됩니다
- box-sizing: border-box를 사용하면 패딩과 보더가 전체 너비에 포함됩니다

핵심 요소 태그

  1. 폼(Form)은 사용자 입력을 받아 서버로 전송하는 HTML 요소입니다
  2. < form> 태그로 입력 요소들을 감싸고, action과 method 속성을 설정합니다
  3. < input> 태그로 다양한 형태의 입력 칸을 만들 수 있습니다
  4. < button type="submit">으로 폼 데이터를 전송합니다
