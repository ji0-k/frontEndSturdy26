'WWW'

웹이 World Wide Web이라고 불리는 이유는 바로 링크(Link) 때문입니다.
거미줄처럼 모든 웹페이지가 서로 연결되어 있어서 사용자가 자유롭게 이동할 수 있죠.

'a 닻'

HTML에서 링크를 만드는 태그가 바로 < a > 태그입니다. 'a'는 'anchor(닻)'의 줄임말로,
배가 항구에 닻을 내리듯 특정 위치로 이동할 수 있게 해준다는 의미입니다.

< a href="이동할 주소" > 클릭할 텍스트 < /a >

'href'

'hypertext reference'의 줄임말로, "어디로 이동할 것인가"를 지정합니다.

'링크의 종류'

  1. 외부 링크: 다른 웹사이트로 이동 (https://www.google.com)
  2. 내부 링크: 같은 사이트 내 다른 페이지로 이동 (about.html)
  3. 앵커 링크: 같은 페이지 내 특정 위치로 이동 (#section1)

'기본링크만들기'
