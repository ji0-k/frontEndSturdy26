핵심개념

  • < table> : 전체 테이블을 감싸는 컨테이너(골조역할)
  • < thead> : 테이블 머리글 영역 , 제목 행들을 묶어줌
  • < tbody> : 테이블 본문 , 실제 데이터들어가는곳
  • < tr> : table row , 테이블의 한 행
  • < th> : table header , 제목 셀 , 굵게 표시되고 가운데 정렬
  • < td> : table data , 일반 데이터 셀

의미론적으로 명확한 구조르 만들어 스크린 리더나 검색엔진이 데이터의 관계를 정확하게 파악하기위함
