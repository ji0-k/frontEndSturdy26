반응형 중단점

부트스트랩은 5가지 화면 크기를 기준으로 지원

  • xs (Extra Small): < 576px (스마트폰)
  • sm (Small): ≥ 576px (큰 스마트폰)
  • md (Medium): ≥ 768px (태블릿)
  • lg (Large): ≥ 992px (데스크톱)
  • xl (Extra Large): ≥ 1200px (큰 데스크톱)

그리드 시스템

그리드 시스템의 3요소

Container

전체 내용을 감싸는 컨테이너

- .container: 고정 너비 (반응형 중단점에 따라 변화)
- .container-fluid: 화면 전체 너비

Row

수평으로 배열되는 컬럼들의 그룹

- .row 클래스 사용
- 컬럼들을 가로로 묶어주는 역할

Column

실제 내용이 들어가는 영역

- .col_* 클래스 사용
- 12개 중 몇개를 차지할 지 설정
