문자열 조작 실습

개인화 메시지 생성기

자주 발생하는 오류와 해결책
- 오류 : 백틱() 대신 작은따옴표(') 사용
- **해결** : 키보드에서~키를 Shift 없이 누르면 백틱입니다
- 오류 : ${}` 안에 변수명 오타
- **해결** : 변수명을 정확히 확인하고 콘솔에서 디버깅하세요

문자열다루기 핵심

문자열 생성: 작은따옴표, 큰따옴표, 백틱 세 가지 방법
템플릿 리터럴: 백틱과 ${}를 사용한 동적 문자열 생성
문자열 연결: + 연산자보다 템플릿 리터럴이 더 편리
실무 활용: 사용자 개인화 메시지, 동적 콘텐츠 생성에 필수
